# Text-Anonymizer
Инструмент для автоматической замены конфиденциальных данных (Email, номера телефонов, имена) в тексте на заглушки
. Стандартный запуск (демо-режим)
Программа возьмет встроенный тестовый текст и покажет результат:
bundle exec ruby main.rb
2. Анонимизация своего текста
Просто передай нужную строку в кавычках после названия файла:
bundle exec ruby main.rb "Мой номер 89991234567, пиши на secret@mail.com"
